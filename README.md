## 🚀 快速开始

### 示例一：小米平台

查询号码 `02039575218`：
```bash
curl "https://tmini.net/api/haoma?phone=02039575218"
```
返回结果：
```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "phone": "02039575218",
    "tag": "广告推销",
    "tag_type": "marked",
    "count": 1281,
    "logo": "",
    "is_scam": true
  }
}
```
